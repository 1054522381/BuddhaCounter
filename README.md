## 主要功能 (Features)

- 1、计数与统计：记录每天（支持分时段记录）念佛次数并自动统计当天，当月及总次数；
- 2、图表展示：数据变化一目了然；
- 3、计划提醒：设定每日计划，自动提醒；
- 4、念佛计数器：在手机上进行计数，自动保存记录；
- 5、支持`Android 4.1 (Jelly Bean)` ~ `Android 6.0 (Marshmallow)`
